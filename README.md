# Spring CRUD Project

## Description
This project is a **Spring Boot web application** that implements **CRUD (Create, Read, Update, Delete) operations** for user management. It demonstrates **Spring Bootâ€™s MVC architecture**, **RESTful APIs**, and **database interaction** with MySQL. This project is ideal for learning backend development and full-stack application concepts using Java and Spring Boot.

## Features
- Create, Read, Update, Delete users via web interface or APIs.
- RESTful backend APIs for CRUD operations.
- MySQL database integration for persistent data storage.
- Exception handling for robust and reliable operations.
- Easily extendable for larger applications.

## Technologies Used
- Java 8+
- Spring Boot
- MySQL
- Maven / Gradle
- HTML & Thymeleaf (optional frontend)
